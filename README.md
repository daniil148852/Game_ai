# Face Analyzer AI

A beautiful Android application that uses AI to analyze faces in real-time using the camera.

## Features

- **Real-time Face Detection**: Detects faces using ML Kit
- **Emotion Recognition**: Identifies emotions (Happy, Sad, Angry, Surprised, Neutral)
- **Face Quality Analysis**: Evaluates face quality based on pose, brightness, and sharpness
- **Symmetry Score**: Calculates facial symmetry
- **Skin Condition Analysis**: Analyzes skin texture, tone, and clarity
- **Face Mesh**: Optional 3D face mesh overlay with 468 points
- **Beautiful UI**: Modern Material Design 3 with gradient backgrounds and smooth animations

## Tech Stack

- Kotlin
- Jetpack Compose
- CameraX
- ML Kit Face Detection
- ML Kit Face Mesh Detection

## Requirements

- Android SDK 24+
- Camera permission

## Getting Started

1. Clone the repository
2. Open in Android Studio
3. Build and run on a physical device (camera required)
